Support & Help

HouseLearning Help Center

Need answers? You’re in the right spot. This Help Center covers the most common questions, fixes, and how-to guides for everything across the HouseLearning ecosystem.

1. What Is HouseLearning?

HouseLearning is a lightweight learning platform designed to run fully on static files. No databases, no backend headaches, no locked-in systems — just clean HTML, CSS, JS, and community-powered docs.

2. How Do I Fix Common Errors?

Here are the top issues people hit:

  • Page not loading: Clear cache or check GitHub Pages build status.
  • 404 links: Remember GitHub Pages is case-sensitive.
  • Styles missing: Ensure Tailwind CDN loads before custom scripts.
  • Feedback widget not showing: Your browser might be blocking third-party scripts.

3. Account & Profile Support

If you're having issues logging in or managing your profile:

  • Double-check your email spelling.
  • Try resetting your password.
  • If you're using school Wi-Fi, scripts might be blocked — try mobile data.

4. Need Direct Help?

If none of this solves your problem, you can just hit the Contact Support page or open a GitHub issue with detailed steps, so the team can reproduce your problem.
