Introduction

Overview

HouseLearning Pages is a free, safe, and beginner-friendly way to publish static websites. It’s built on top of GitHub Pages and powered by SafeLibrary — a system designed to keep uploads secure, predictable, and abuse-resistant.

What Is HouseLearning Pages?

HouseLearning Pages allows users to upload static website files and publish them instantly under the pages.houselearning.org domain.

There are no servers to configure, no databases to manage, and no backend code to maintain. If it runs in a browser, it works here.

What Is SafeLibrary?

SafeLibrary is the security and validation layer behind HouseLearning Pages. It controls what files can be uploaded, how they’re stored, and how they’re deployed.

  • Blocks dangerous file types
  • Enforces static-only hosting
  • Prevents execution of uploaded code
  • Protects the hosting infrastructure

Who Is This For?

  • Students learning web development
  • Educators publishing learning content
  • Developers hosting demos or documentation
  • Anyone who wants simple, safe static hosting

This platform prioritizes learning, safety, and simplicity over unlimited power.

How It Works (High Level)

  1. You upload static files
  2. SafeLibrary validates them
  3. A GitHub Action deploys the site
  4. GitHub Pages serves it globally

No manual deployment steps required.

What This Platform Is Not

  • Not a backend hosting service
  • Not a database provider
  • Not a general-purpose file host
  • Not a replacement for VPS hosting

These limitations are intentional — they keep the platform fast, free, and secure.
