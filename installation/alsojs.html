Also.js Plugin Documentation

Overview

Also.js is a JavaScript plugin that integrates with HouseLearning to provide:

  • Firebase authentication tracking
  • DevTools detection & logging
  • Console paste tracking
  • Full-page translation using Google Translate API
  • Single-page logging (once per page forever) to Firebase

Installation

Include the plugin in your HTML:

<script type="module" src="https://www.houselearning.org/also.js"></script>
